# AI Agent Framework

This project provides a foundational Streamlit application framework for an AI Agent project. It includes essential UI components, data handling capabilities (CSV/Excel upload and visualization), and environment setup for a clean and extensible structure.

## Features

*   **Streamlit Application:** A user-friendly web interface built with Streamlit.
*   **Navigation:** Sidebar navigation for "Home", "Data Explorer", and "AI Agent Interaction" sections.
*   **Environment Variables:** Configuration loaded from a `.env` file.
*   **Logging:** Basic application logging to console and `logs/app.log`.
*   **Data Explorer:**
    *   Upload CSV or Excel (`.xlsx`) files.
    *   Preview uploaded data in an interactive table.
    *   Generate various interactive charts (Bar, Line, Pie, Scatter) using Plotly based on selected columns.
    *   Basic error handling for file uploads and chart generation.
*   **AI Agent Interaction (Placeholder):** A dedicated section for future integration of AI agent interfaces.
